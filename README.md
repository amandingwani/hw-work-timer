# hw-work-timer
Countdown timer with LED display driven by TM1637 display module and Arduino.

## TM1637 connections:
CLK - Pin 4 of Arduino
DIO - Pin 3 of Arduino
VCC - 5V  of Arduino
GND - GND  of Arduino

# Led and Button connection:
LED_PIN 5
BUTTON_PIN 2
